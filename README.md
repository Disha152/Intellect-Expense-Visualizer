# GraphQL Crash Course - Build an Expense Tracker App

![Demo App](https://i.ibb.co/WHyMscm/Screenshot-42.png)

This project is designed to help you manage your expenses efficiently using a modern tech stack .

## Features:

-   🌟 Tech stack: MERN (MongoDB, Express.js, React.js, Node.js) + GraphQL
-   📝 Learn type definitions and resolvers for defining GraphQL schema and data fetching logic
-   🔄 Mutations for modifying data in the GraphQL API and establishing graph relations
-   🎃 Authentication with Passport.js and MongoDB session store
-   🐞 Error handling both on the server and on the client
-   ⭐ Deployment made easy with a platform called Render


### Setup .env file

```js
MONGO_URI=...
SESSION_SECRET=...
```

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
